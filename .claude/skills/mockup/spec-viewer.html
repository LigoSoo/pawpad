프로젝트 폴더를 선택하세요

데이터 파일(prd.json · fts.json · userflow.json · wire.json)이 들어있는 폴더(예: src/viewer/)를 1회 선택하면, 4탭이 자동으로 로드됩니다. 편집 후 저장하면 같은 파일에 제자리 덮어쓰기됩니다(다운로드·백엔드 없음). 재로드로 반영 확인.

PRD 문단. ✎수정 / ×삭제 · 파일에 남은 항목 = 설계/개발 대상(삭제=제외, 별도 승인 없음).

메뉴 → 기능그룹 → 스팩. ✎수정 / ×삭제 · 남은 항목 = 개발 대상. 상태(읽기전용, agent 갱신): 예정 진행중 완료 ·

메뉴 구성도(앱→탭→화면→하위). 드래그(⠿)로 순서·위치 이동 · +하위 / ✎수정 / ×삭제 ·

화면별 lo-fi UI(컴포넌트 스택, agent 생성). 검토·조정만: ↑↓순서 · ✎수정 · ×삭제 · 우상단 프레임(mobile/web) 토글.
